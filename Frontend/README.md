# 🏛️ Heritage Site Booking (HSB) - Frontend

A modern and responsive Heritage Site Booking web application built using **React.js**. The platform enables users to explore famous heritage places, view details, book tickets, select time slots, and complete secure bookings through an intuitive interface.
# ✨ Features

- 🏛️ Browse popular heritage places
- 🔍 Search & Filter destinations
- 📍 View detailed place information
- 🎟️ Ticket Booking System
- 👥 Visitor Details Form
- 💳 Payment Option
- 📱 Fully Responsive UI
- ⭐ Recommended Places Section
- 📜 Booking History
- 🚀 Fast Navigation using React Router
- ### State Management

- React Hooks
- useState
- useEffect

### Routing

- React Router v6

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Jaydip-DataAnalyst/HSB.git
```

Go to frontend

```bash
cd HSB/Frontend
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

or

```bash
npm start
```
